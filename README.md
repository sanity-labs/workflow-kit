# workflow-kit

Reusable workflow engine and UI primitives for Sanity-powered applications.

## Installation

```sh
pnpm add @sanity-labs/workflow-kit
```

## Usage

Import the pieces you need from the package entrypoints:

```ts
import { workflowRoleSlugMatches } from "@sanity-labs/workflow-kit";
import { WorkflowStatusPath } from "@sanity-labs/workflow-kit/react";
import { StatusPathInput } from "@sanity-labs/workflow-kit/studio";
```
