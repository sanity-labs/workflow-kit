# sanity-workflow-kit

Reusable workflow engine and UI primitives for Sanity-powered applications.

## Installation

```sh
npm install sanity-workflow-kit
```

## Usage

Import the pieces you need from the package entrypoints:

```ts
import {workflowRoleSlugMatches} from 'sanity-workflow-kit'
import {WorkflowStatusPath} from 'sanity-workflow-kit/react'
import {StatusPathInput} from 'sanity-workflow-kit/studio'
```
